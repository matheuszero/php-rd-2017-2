Array_search

O Array_search,procura por um valor em um array e retorna sua chave correspondente caso seja encontrado.
Ele pode retornar um valor booleano FALSE , ou um valor não booleano “0” , que pode ser entendido como FALSE.

Sintaxe

array_search(value,array,strict)

  • Value: Especifica o valor que vai ser pesquisado.
  • Array: Especifica a matriz.
  • Strict é opcional. Se este parâmetro é definido como TRUE, esta função irá procurar elementos idênticos na matriz.
