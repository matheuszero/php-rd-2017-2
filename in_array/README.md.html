in_array

O in_array,checa se um valor existe em um array. Se existir retorna true.

Sintaxe

in_array(search,array,type)

  • Search:Especifica o que procurar.
  • Array:Especifica a matriz de pesquisa.
  • Type:É opcional, se é definido como TRUE, os in_array() procura o tipo específico na matriz.
